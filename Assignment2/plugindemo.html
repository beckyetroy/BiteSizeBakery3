About my JQuery Plugin

I chose to incorporate slide and push menus into my website. I chose this plugin because I thought it looked visually appealing and could make for an interesting feature.

Slide menus are a set of fixed menus that will slide out from any of the edges of the page when prompted.
Push menus are slide menus that simultaneously push the body of the page to the left or right, as if the menu is pushing the page.

How I used it

I used the Left Push menu as the main navigation menu for my website. I also used the Bottom Slide menu to display links to social media accounts in the 'contact us' page.
